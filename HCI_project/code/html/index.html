Πληροφορίες

  • Για να μεταβείτε στο κύριο μέρος του συναγερμού πατήστε το κουμπί 'Συναγερμός'
  • Για να δείτε τις ζώνες του συναγερμού πατήστε το κουμπί 'Ζώνες'
  • Για να μεταβείτε στις ρυθμίσεις του συναγερμού πατήστε το κουμπί 'Ρυθμίσεις'. Διαθέσιμες ρυθμίσεις: Αλλαγή κωδικού συναγερμού, Αλλαγή Ονόματος Ζώνης, Αλλαγή χρονοκαθυστέρησης εξοπλισμού

Δώστε τον κωδικό πρόσβασης προκειμένου να μεταβείτε στις ρυθμίσεις
