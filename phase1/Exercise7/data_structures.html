Βασικές Δομές Δεδομένων της Python

List

Η Λίστα χρησιμοποιείται για την αποθήκευση πολλών αντικειμένων σε μία μεταβλητή. Η λίστα επιρέπει τα διπλώτυπα και το μέγεθος της μπορεί να αυξομειώνεται.

List Initialization

>>> numbers = [1,2,3,4]
>>> numbers
[1, 2, 3, 4]

Αλλαγή αντικειμένου σε συγκεκριμένο index

>>> numbers[0] = 9
>>> numbers
[9, 2, 3, 4]

List iteration

>>> for x in numbers:
        print(x)
    
        
9
2
3
4

Προσθήκη αντικειμένου στην λίστα

>>> numbers.append(5)
>>> numbers
[9, 2, 3, 4, 5]

Διαγραφή αντικειμένου στην λίστα προσδιορίζοντας το index

>>> numbers.pop(0)
9
>>> numbers
[2, 3, 4, 5]

Διαγραφή αντικειμένου στην λίστα προσδιορίζοντας την τιμή του στοιχείου

>>> numbers.remove(5)
>>> numbers
[2, 3, 4]
Method Description
append() Adds an element at the end of the list
clear() Removes all the elements from the list
copy() Returns a copy of the list
count() Returns the number of elements with the specified value
extend() Add the elements of a list (or any iterable), to the end of the current list
index() Returns the index of the first element with the specified value
insert() Adds an element at the specified position
pop() Removes the element at the specified position
remove() Removes the first item with the specified value
reverse() Reverses the order of the list
pop() Sorts the list

Set

Το σύνολο είναι παρόμοια δομή δεδομένων με την λίστα αλλά με την διαφορά ότι στο σύνολο δεν επιτρέπονται τα διπλότυπα

Set Initialization

>>> set = {1,2,3,4,5,5,5}
>>> set
{1, 2, 3, 4, 5}
>>> #(Βλέπουμε ότι δεν
>>> υπάρχουν διπλότυπα)

Set iteration

>>> for x in set:
        print(x)

	
1
2
3
4
5

Προσθήκη αντικειμένου στo σύνολο

>>> set.add(6)
>>> set
{1, 2, 3, 4, 5, 6}

Διαγραφή αντικειμένου άπό το σύνολο

>>> set.remove(6)
>>> set
{1, 2, 3, 4, 5}

Union

>>> set ={1,2}
>>> set2 = {2,3,4} 
>>> set3 = set.union(set2)
>>> set3
{1, 2, 3, 4}

Intersection

>>> set ={1,2}
>>> set2 = {2,3,4}
>>> set3 = set.intersection(set2)
>>> set3
{2}

Dictionary

Τα λεξικά χρησιμοποιούνται για να αποθηκεύσουν δεδομένα της μορφής κλειδι:τιμή και δεν επιτρέπουν δεδομένα με διπλότυπα κλειδιά.

Dictionary Initialization

>>> dict = {1:"Mary",2:"Paul",3:"Tyler"}
>>> dict
{1:'Mary',2:'Paul',3:'Tyler'}

Εύρεση τιμής για συγκεκριμένο κλειδί

>>> x = dict.get(1)
>>> x
'Mary'

Εύρεση όλων των κλειδιών του λεξικού

>>> x = dict.keys()
>>> x
dict_keys([1, 2, 3])

Προσθήκη στοιχείου στο λεξικό

>>> dict[4] = "Josh"
>>> dict
{1:'Mary',2:'Paul',3:'Tyler',4:'Josh'}

Έλεγχος ύπαρξης κλειδιού στο λεξικό

>>> if 1 in dict:
	print("Yes, key 1 exist")
       	
Yes, key 1 exists

Επεξεργασία λεξικού

>>> dict.update({2:"George"})
>>> dict
{1:'Mary',2:'George',3:'Tyler',4:'Josh'}

Διαγραφή στοιχείου με συγκεκριμένο κλειδί

>>> del dict[4]
>>> dict
{1:'Mary',2:'George',3:'Tyler'}

Dictionary Iteration

>>> for x in dict:
	print(dict[x])

Mary
George
Tyler
